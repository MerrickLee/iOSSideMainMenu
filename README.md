## Example

To run the example project, clone the repo, and run `pod install` from the Example directory first.
All you need to do is drop iOSSideMainMenu files into your project.

## Requirements
* Swift3 or higher
* Xcode 8 or higher
* iOS 8.0 or higher

## Installation

iOSSideMainMenu is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "iOSSideMainMenu"
```
## Demo
Build and run the iOSSideMainMenu project in Xcode to see JMSideMainMenu in action.
